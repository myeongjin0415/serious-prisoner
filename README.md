- 주제

    - 지식의 축적과 이해의 가치.
- 스토리
    - **사형수**
        - **기억을 잃은 사형수.**
            - 이렇게 해야지 행동할 동기가 생긴다.
        - 선한 사형수. 독재 정권에 저항했다던지,
            - 독재 정권에 저항한다로 가면 → 스파이와의 접선 같은 요소를 넣을 수 있다.
                - 탈출엔딩이 트루엔딩이다.
                - 스파이가 숨어있는게 후반에 큰 단서로서.
                - 탈출하기 위해서는 시간 + 공간 + 특정 도구를 마련하면 탈출 가능.
